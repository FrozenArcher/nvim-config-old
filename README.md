# nvim-config

neovim configuration

## needed packages

* (aur) nvim-packer-git
* nodejs
* npm
* universal-ctags
* clang
* stylua


## main components

* plugin manager: packer.nvim
* LSP: completion: coc.nvim
* colorscheme: neon
* file explorer: nvim-tree
* tabline: bufferline
* statusline: lualine
